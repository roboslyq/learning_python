# crawl爬虫的简单步骤
## 不需要登录的网站(html内容)
1、确定网页的主页（入口路径）

2、通过入口路径获取返回菜单栏

3、循环遍列菜单栏，获取所有的菜单

3.1 遍列菜单时，通过标签获取content的内容

3.2 将内容保存到自定义的html模板中

3.3 将相对路径转换为绝对路径

3.4 将html转换为pdf

4、合并不同的pdf

## 需要登录的网站
1、确定网页的主页（入口路径登录页）

2、登录操作(post + json)，获取对应的cookie及session信息

3、获取相应的菜单（请求带上cookie和session信息）

3.1 遍列菜单时，通过标签获取content的内容

3.2 将内容保存到自定义的html模板中

3.3 将相对路径转换为绝对路径

3.4 将html转换为pdf

4、合并不同的pdf
